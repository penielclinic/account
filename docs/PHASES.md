## 우선순위 & MVP

**MVP (출시 가능 최소 기능)**: Phase 0 + 1 + 2 = 약 8주차

이 시점부터 기존 엑셀 방식 대비 충분히 유용하므로, **8주차에 1차 배포** 후 Phase 3~5는 점진적으로 추가하는 것을 권장합니다.

| 단계 | 누적 기간 | 핵심 산출물 |
|---|---|---|
| Phase 0 | 1주차 | 개발 환경 셋업 |
| **Phase 1** | **5주차** | **수동 입력 회계 작동** ✓ |
| **Phase 2** | **8주차 (MVP)** | **AI 입력 + 이음 연동** 🚀 |
| Phase 3 | 11주차 | 비교 대시보드 |
| Phase 4 | 13주차 | 자연어 AI 질의 |
| Phase 5 | 15주차 | 보고서 + 정식 출시 |
